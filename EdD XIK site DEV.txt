7 Running the Development Environment

Open terminal.

Go to project:

cd K:\doctoral-progress

Install dependencies if needed:

npm install

Start dev server:

npm run dev

Open browser:

http://localhost:3000

The dev server hot-reloads automatically.

8 PowerShell Script Error Fix

If you see:

npm.ps1 cannot be loaded because running scripts is disabled

Run:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

Then run:

npm run dev

10 Publishing Changes

The project uses GitHub → Vercel automatic deployment.

Your automation script:

commit.bat

Location:

K:\doctoral-progress\commit.bat
Deployment Steps

Run:

commit.bat

Script performs:

1 Stage all changes
2 Commit with message
3 Push to GitHub

After push:

Vercel automatically deploys the new version

Deployment usually completes in:

30–90 seconds
11 Checking Deployment

Open:

https://vercel.com/dashboard

Look for latest deployment.

Status should be:

Ready

If error occurs, open deployment logs.

16 Working With AI Assistants (Claude / ChatGPT)

When asking an AI to modify the site:

Provide:

file path
existing code
desired change

Example prompt:

Update src/app/(protected)/dashboard/page.tsx

Add a section listing EDUC 830 papers.
Keep the existing Tailwind styling pattern.
Do not modify other components.

This prevents accidental structural breakage.
